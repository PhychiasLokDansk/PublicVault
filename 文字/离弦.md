# THE BAG-RIDER
## **骑包者**
*——现实的物事总显得骨感，所以人们总喜欢想象。*
- 这天在我磨完21道大题的圆锥曲线专项强化，又对付掉两张物理力学练习后，我听到一阵嘈杂的声音，开始是零碎的咒骂声，然后有玻璃破碎的声音，我便知道又有人犯起了癔症。
- 班里好事的哥们听到响声就跑出去了，但这时段长来了，把他们都赶回了教室。我把耳机摘下藏好，继续写我的物理练习。
- 后来大约是保卫科的人来了，教室外低语声，痛苦的呻吟声，脚步声相互交错。班里最好事的那家伙仍不死心，趴在窗台看，不知道他看到了什么。
- 我也想知道外面发生了什么，但我还有作业要写。
- 在我的物理自编还剩最后一道大题时，我听见很响的消防车的警笛，应该是消防队员来了。我原以为这警笛声会掠过学校向山上去，去解救山上被大火围困的动物，但它在学校停车场停下了。
- 第二天吃午饭的时候，有一位九班的同学眉飞色舞地描述了当时的场景。
- 他当时正在后柜站着订正练习——坐久了人会太僵硬，许多同学喜欢晚自习时不时站着写会作业，也好清醒一下容易打瞌睡的自己。
- 事起猝然，他听得咒骂声，回头一看，这位仁兄已身在半空，这一脚来势很快，却偏偏踹了个空，被踹的同学都还没反应过来，那玻璃已碎了。
- 这位仁兄大约是个练家子，否则便很难一脚踢破门上的钢化玻璃，还把脚卡在里面。而且碎掉的玻璃真的能将一个人的大腿完全卡在里面吗？
- 消防队员冲上来的时候那个同学脸上的表情想必十分精彩。消防队员是怎样敲碎门上的玻璃，剥离他的腿而不伤到他的呢，为什么第二天除了九班主教室后门上消失的玻璃以外，这整件事什么痕迹也没留下呢？
- 这件事情，会不会是我臆想出来的呢？包括那个九班同学的描述，是不是都是我自己臆想出来的呢？我为什么会臆想出这样的事情来呢？
- 第二天早上，关于这件事沸沸扬扬的讨论打消了我的疑虑。
- 一时间似乎没有别的什么新闻了，大家都在讨论这个事情，这事情实际上是关于一个人的癔症的——武侠小说看多了会得的那一种癔症，他大约觉得自己猝起一脚，对方这样十恶不赦之人，不死也残。
- 但其实那一脚究竟是为了什么？这并没有人太关心，这不是我们的风格，这件事情的起因经过结果并不比鲜血和碎玻璃有意思，不是么？至于现在这位仁兄是否已经缝完了针，回到学校来上课，更没有人关心了。
- 到了下午这件事情好像又没有人再关注了，这件事情原不是什么值得深挖的事情。但是热度这样快地消退使我心里生出怅然的感觉——我又开始觉得这是一件根本不存在的事情了，可我的大脑为什么要编造一个这样的故事来欺骗我自己呢，为什么不能编造一些小说般引人入胜的情节来让我好好地放松自己呢？
- 我回到家，写完了英语的一篇随堂作文，准备上床睡觉。
- 这时候我在书桌前听到一阵轰鸣，我想，这时候从我头顶路过的飞机要去哪里，机翼上红色的绿色的灯一闪一闪，在这夜色中我昏睡过去。
- 梦里，我骑着一个大号的SWISSTWIN背包飞了起来，这包作为一个坐骑好到了极点，我两腿从原本应该挎在我肩上的背包带下穿过，手紧紧地攥着背包的提手。我一夹膝盖，空荡荡的书包轻飘飘地载着我飞起来了。
- 我骑着我的包，晃晃悠悠地来到了五楼的教师办公室。我抬腿跨进过道，然后把包拿了进来。我敲了敲办公室的门，里面有一个面熟的老师，但我怎么也想不起来他的名字，但是我大声地喊：“老师！”
- 老师抬头望了我一眼，又低下头去改他的卷子。
- 我绕到窗户那边，用力地敲窗户，大喊：“老师！”
- 老师不理我，我拧了拧门把，发现门锁上了，于是我又绕到窗户这，打开窗户，扒住窗台，蹬着墙，翻了进去，在其中一个办公桌上找到了今明两天的作业，又看见后天的作业静静地躺在长桌上。我想了很久，要不要把后天的作业一起拿走。
- 突然我发现，那个老师坐的位置上已经没有老师了，取而代之的是一人高的作业，好多好多的作业，我数了数，不仅三天后的作业，接下去一周整个年段的作业，堆满了这个破旧的椅子。
- 我惊出一身冷汗，手不由得攥紧了装满作业的、沉重的书包。我转身想走，转身的那一瞬间发现背后出现了好多作业，堆起来高出了我的头许多，我于是转向另外一个方向的出口，发现到出口的道路上都是作业——我无路可走了。
- 我开始试图攀登作业组成的小山，正当我爬到一半时，它们坍塌了，它们直直地塌进了地里，把爬到许高的我重重摔在了地上。我匆忙爬起来，捡起掉在一旁的包，翻过窗户，翻出栏杆，把包放在空中。
- 当我把跨上我的包的那一刻我发现，装满作业的包，很沉重，它不再轻飘飘地了，它开始时下降的不快，但两秒钟不到就失去了控制，我和包一起自由落体，重重地摔在了地上。
- 我疼痛不已。
- 我醒来，睁开眼，发现阳光不偏不倚地照在床头闹钟，它没有响，阳光却亮的吓人。

# THE BARRIER
## 套
- 每到冬天的时候我就喜欢穿一件黑色的大衣，厚厚的黑色大衣，再穿一条黑色的裤子，厚厚的黑色裤子，黑色裤子下的鞋子也是黑色的，走在风里大衣的下摆会随风飘动，慢慢走路时一点声音也没有，到了晚上无灯的地方就没有人看的见我，我就是阴影里的幽魂。
- 戴上兜帽的我看不清远方的路，戴上耳机的我听不见周围的人，周遭的世界对我来说仿佛不存在。黑色的我对于周遭的世界来说也仿佛不存在。此刻我存在着，却隐形如空气。这并不使我感到难受，反而像是一种高尚的追求。
- 我喜欢穿成这样在黑夜里的操场走着。夜里没灯的操场偶尔能看到两三点星光，如果是夏夜晴空或许能看到火红的木星。但是现在是冬天，阴阴的天空，刚刚才下过雨，操场上有小水洼，小水洼倒映着思诚楼的灯光，思诚楼的灯火通明。课间大家都围着操场绕圈，像马戏团里的动物，一圈又一圈，有些人在慢跑，有些人在散步。
- 我在操场中央坐下来，一边听巴赫的二部创意曲，一边看他们绕圈，像在装逼。
- 装逼这个事情很有趣，因为装逼这件事情本来可以是有意识的，也可以是无意识的，毫无疑问每个人都有意无意的要装一下逼，因为装逼这件事情本来是很爽的，装逼带来的满足感和优越感，很少有人能抗拒。
- 然而装逼是常常失败的，这或许因为装逼犯们往往没有与之相配的实力，又或许因为装逼犯们总是喜欢忽略客观条件，过于相信主观意识的能动性了。
- 我似乎很喜欢装逼，其实我并不太清楚我是不是很喜欢装逼，以及我为什么装逼，但他们叫我逼神，那我一定是装逼装的厉害了。仔细想想，也许我确实没有与我吹出的牛逼相匹配的实力。但很多时候我意不在装，只是想说就说。说的话的内容也许总是有些扎耳，不那么动听，总是要给人一种居高临下的感觉。这我很久以后才意识到。
- 也是到了很久以后，也就是现在，我才堪堪学会闭上嘴巴。这可不可以作为我选择成为套中人的原因？又或者大家都是套中人，又或者根本没有套中人？
- 我想把这个事情给阿震讲讲，但是我想阿震可能不是很理解这件事情，因为他显然不是一个喜欢装的人。这件事情原本也不是什么大事，讲给他听不过是给我自己寻求一点倾诉带来的轻松感罢了。
- 我想阿震是天生的套中人，他不需要厚厚的黑色大衣，他也是一个套中人，但他套住的东西不是我套住的东西，我们不是同一种套中人。就好像被网兜住的鱼，和被坝拦住的水，是不一样的。
- 我想要套住的不是我所能套住的，而他套住的本不是他需要套住的。也许我们在这一点上相反的恰到好处，所以我们成为了朋友。
- 在操场上我也常听听别的歌，有时候听听Adele，有时候听听LinkinPark，有时候听听周杰伦，有时候听听阿震讲话。
- 我喜欢的东西和阿震喜欢的东西不尽相同，但是我总喜欢跟他说说话，跟他说话的时候像在跟自己说话，跟我自己说话的时候又像是在跟阿震说话，我想阿震大概也是这样觉得的。
- 有一段时间操场上常来一只橘色狸花，年纪很小且很活泼的一只小猫，虹膜是绿色的，有棕色的菱形花纹，在月光下是澄澈的鹅黄。
- 看到猫我总是很高兴，每一只猫都是天生在套中的，至少对人来说是在套中的。
- 有时候我们会对视，有时候我们对舞，以猫的方式对舞，这看起来有些怪异的游戏在那些绕圈的人看来是不是在发疯呢，这问题总是萦绕在我的心头，我想了一下，决定不去想它，只要橘色的小猫不觉得我在发疯，我就没有发疯。疯的不是我，疯的是冬天的风。
- 橘猫的毛很软，脏脏的，但是很软，我决定叫这只猫阿熏，因为它总喜欢在花丛里蹭蹭，沙坑里蹭蹭，身上沾染了各种奇奇怪怪的味道，有一种说不出的奇怪感觉。和我被染成五颜六色的套子有点像，或许我应该改名叫做阿染。我觉得阿熏应该是认得我的，我总是希望我见到的猫能认得我，我也在尽力认得它们，尽管分辨另外一个物种个体差异的技巧并不是随随便便就能掌握的。
- 阿熏的花纹不好看，阿熏不是一只漂亮的强壮的猫，但是是这操场上唯一的猫，是操场上我知道的第三个在套中的生物。也许操场上空飞得跌跌撞撞的蝙蝠也是在套中的，但我想了一下，蝙蝠这种生物岂不是本来就很聒噪的，飞的时候大声地叫喊，然后靠反射回来的声波来辨别方向。他们并非真的安静，只是我们听不见他们的声音罢了。即便他们昼伏夜出，这样聒噪的生物绝不是在套中的。
- 套中的人似乎很难知道自己是不是在套中的，套中人和套中人本是不易相遇的，套中人更没有与另一个套中人相恋的本领。
- 他们说阿熏是一只猫，又说猫是不会信任人的。其实我也知道猫是在套中的生物，但是我却总是割舍不下对猫的喜欢。
- 阿雯觉得猫很讨厌，她觉得猫的眼神太凌厉，她觉得猫看起来总像是打算一口吃掉她。
- 她不喜欢猫，更不喜欢猫的眼神。可是猫的眼神里有什么，人怎么看的清楚，我向猫的眼睛里看去，总是只看到我自己的倒影，就好像看向操场上的小水坑。
```
我踩过操场上的小水坑
	操场上的小水坑
	在雨季也一样长不出浮萍
雨季过去以后
	它在风里沉沉睡去
```

# THE MOONLIGHT
## 月光
- 每天的放学，我匆匆收好书包冲下楼梯，冲进地铁站，就为了那一班22：02分的地铁。然而今天我睡着了，在晚自习结束前的最后几分钟，我趴在课桌上睡着了，口水浸湿了我的生物习题集，我醒来时，已然是22：02 。
- 脑袋发涨着，我挣扎着起身收拾杂乱的课桌和小山般的习题册。烦躁在身上蔓延，身旁同学的打闹声令我倍感心烦——他们的快乐此刻与我的痛苦同在，像是对我的一种嘲讽。
- 踱下六层楼，磨磨蹭蹭地走出校门，右转来到地铁口，乘电梯下行到地铁里。平时的安检口总是排着长长的队伍，而今天我下来的确实是太晚了，安检口一个人也不剩了。
- 在门闸打开的提示音中，我走进地铁车厢，环顾四周，我挑了个没人的地方坐下，打开kindle，戴上耳机，我什么都听不见，此刻外界与我就毫无联系了，我与我的灵魂相依偎，我看之电子屏上灰白的文字，慢慢陷入一种恍惚，看到动情处不禁鼻头一酸。
- 抬起头来，惊觉对面坐着一个女孩，拿着一台荣耀5x手机，戴着条白色的耳机在听着什么。
- 她身上也穿着全市统一的校服，看起来和我差不多年纪，裤脚一看就是改过的，马尾扎得很干净。中山公园站，车门打开，铃声响起，她在这时笑了，盯着那块小小的对角线5英寸左右的LED屏幕笑了，屏幕上的光映着她的脸苍白，青蓝色的血管在她的脖颈浮现。这是一种来自我的世界之外的美。
- 我没太看清她的脸，因为她低头玩着手机，而我并不好意思直勾勾地盯着人家看，只是偶尔用余光瞟一两眼。每个女生天生都带有她们的气场，看不见摸不着，但你知道它在那，和你坐的椅子一样真实。这个女生的气场令我感到有些窒息，空气中满是看不见的桎梏，时间变得慢了下来，好像凝固了一样。
- 红色的警铃声中车门再度打开，这是这趟车的终点站，她站了起来，甩了甩也许因为拿了太久手机而酸痛的手，走出了车厢。我紧紧跟随着，我不是那种能够引人注目的人，所以她不会注意到我。
- 我一路小跑到她前面，赶在她前面上了电梯，我回头看了一眼，想要瞄到她的正脸，她低头看着手机，笑了，想必手机里的东西应该是很有意思的吧。
- 走出了地铁口，我抬头望了望天，天空上除了月亮什么也没有，没有星星，连一朵云都没有，只剩月亮孤零零一个挂在天上。她像一簇火柴点出的小火苗，时间到了就消失得无影无踪。
- 从此我坐地铁总是坐末班的，做地铁的时候也不再看kindle。我假装看着kindle，实则在看她。她弯弯的眼睫毛每次的颤动都与我的心弦共振，她坐在哪里，哪里就有无限的光在散发着。
- 每天从地铁口到家的路变成了我每天期盼的时光，每晚回到家，莲蓬头下的我依然发着呆。
- 如此怅然若失的感觉是爱情吗，这样频率的心弦振动是爱情吗，我反反复复问我自己。
- 终于有一天我决定不再问我自己，我想要去问她，我想知道关于这个问题她有没有答案。还是同样的时间，同样的地铁。在地铁站台上，我忐忑地等待着，在站台里来回踱步。
- 可今天的第一节车厢没有她的身影，我向后面的车厢走去，我在地铁上来来回回，来来回回。地铁加速，减速，加速，减速。我没有看见她，我没有看见月亮。
- 那之后的几天，她也没有再出现在末班的地铁上。
- 我失去了她。
- 很长一段时间里我不知道该如何是好，她也许只是一个幻影，但她在我的生命中出现了。她的出现像钟锤撞击寺院的大钟，震响了整个山谷，而后又归于寂静了。可这余音仍在我脑海里回荡，久久不能散去。
- 深秋的一个夜晚，我在停电的备用教室看到她了。她在窗边望着很远很远的地方，她的头发从长发变成了短发，她的表情如钩月一样悲伤。
- 这使我也悲伤起来了，有一股很呛的味道在我胸中蔓延，我的筋肉纽结，我的灵魂在月光下倍显苍白。她从窗边离开的那一刻，我看见她空洞的眼神，我从这眼睛里读不到任何东西。
- 她从我的身边经过，我想要问她一个问题，但是我无论如何都想不起来这个问题是什么了，我拉住了她，张了张嘴巴，却什么都说不出来，她看着我，就那样安静地看着我，眼里不含有温柔，但有一种美丽的脆弱。
- 当我反应过来时她已经不在了，教室的电来了，灯亮了起来，灯光充盈了整个教室。
